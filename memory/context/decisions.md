## Preferências de Swing Trade de Carlos Dória

1. **Critério Mandatório de RSI:** O conceito de "Elite Entry" ou boa oportunidade de swing trade de exaustão *exige* que o ativo esteja com o RSI Semanal efetivamente sobrevendido (idealmente abaixo de 30).
2. **Rejeição a Sinais Falsos:** Ativos com RSI diário sobrevendido mas semanal esticado (acima de 35-40) são vistos como repiques de curto prazo e não configuram oportunidades macro atrativas para o perfil dele, mesmo que os robôs/scripts os classifiquem como "Elite".
3. **Foco:** Priorizar a identificação de "facas caindo" e "exaustões reais" em tempos gráficos maiores (semanal < 30) ao reportar oportunidades.
