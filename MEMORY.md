| Arquivo | O que contém |
|---------|-------------|
| `memory/context/decisions.md` | Decisões permanentes e irreversíveis |

| `memory/context/business-context.md` | Contexto do negócio, produtos, clientes |
| `memory/projects/zeluna-health.md` | Projeto principal — Zeluna Health |
| `memory/sessions/YYYY-MM-DD-*.md` | Notas diárias (raw capture) |
| `memory/integrations/tools.md` | Mapa de ferramentas, IDs, acessos |
| `memory/feedback/content.json` | Feedback sobre sugestões de conteúdo |
| `memory/feedback/tasks.json` | Feedback sobre execução de tarefas |
| `memory/feedback/tone.json` | Feedback sobre tom e estilo de comunicação |

## 🔄 Ciclo de Memória

```
Sessão (conversa) → memory/sessions/YYYY-MM-DD.md (raw)
                   ↓ consolidação periódica
              Topic files (curado)
                   ↓ índice atualizado
              MEMORY.md (sumário)
```
