## Leveling System

| Nível | Nome | Autonomia | Revisão |
|-------|------|-----------|---------|
| L1 | Observer | Zero — output sempre revisado pelo hub | Cada entrega |
| L2 | Contributor | Baixa — pode sugerir, não executar sozinho | Semanal |
| L3 | Operator | Média — executa dentro de guidelines | Semanal |
| L4 | Trusted | Alta — autonomia quase total | Quinzenal |

### Regras de promoção/rebaixamento
- Promoção via performance review semanal
- Rebaixamento possível se qualidade cair
- NUNCA rushar um agente para L3+ sem histórico comprovado
- Primeiro review: após 1 semana de operação
